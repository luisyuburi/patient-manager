
# Patient Manager

An easy and small project to practice and improve skills.






## Run Locally

Clone the project

```bash
  git clone https://github.com/luisyuburi/patient-manager.git
```

Go to the project directory

```bash
  cd patient-booking-manager
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
